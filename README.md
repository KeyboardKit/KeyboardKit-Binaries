# KeyboardKit Binaries

This repository is used to host binary builds for [KeyboardKit][Repository]. Each tag matches a specific KeyboardKit version tag.

This repository does not add release notes to each tag. Instead, find the corresponding tag in the [main repository][Repository].


## Website

The [KeyboardKit website][Website] has information about KeyboardKit and KeyboardKit Pro, a blog with the latest news and updates, articles, case-studies, pricing and feature information, etc.


## Project

The [KeyboardKit repository][Repository] has a Swift Package that can be used to create keyboard apps for all Apple platforms, as well as a demo app that shows you how to set up ad use KeyboardKit.


## Binaries

The [KeyboardKit Binaries repository][Binaries-repo] is used to store binary arteficts for each version of the KeyboardKit SDK.


## Documentation

The [KeyboardKit online documentation][Documentation] has a getting-started guide, articles for each feature, code samples, etc.
